# Alaktive

Alaktive is a dynamic Todo Card component built to manage tasks with a clean, minimal, and responsive UI. It focuses on state management, smooth interactions, and a distraction-free user experience.

---

##  Overview

Alaktive helps users:
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Organize tasks in collapsible sections

It is designed as a frontend-focused project to practice DOM manipulation, state handling, and UI responsiveness.

---

##  Features

- Dynamic task updates
- Edit mode with smooth UI switching
- Task completion tracking
- Delete functionality
- Collapsible task sections
- Responsive desktop-first layout (landscape optimized)

---

##  Stage 0 Improvements

Compared to the initial version:

- Improved layout structure and spacing
- Introduced proper edit mode flow
- Better state handling for task updates
- Fixed UI responsiveness issues
- Improved interaction feedback (buttons, focus states)

---

##  Design Decisions

- Minimal UI to reduce distractions
- Separate display vs edit modes for clarity
- Used `data-testid` for stable testing and targeting
- Prioritized usability over heavy styling
- Designed for desktop/landscape usage (no vertical scroll focus)

---

##  Accessibility Notes

- Keyboard navigation supported for key actions
- Focus states maintained for interactive elements
- Simple semantic HTML structure used where possible

---

##  Known Limitations

- No backend or database persistence (data resets on refresh)
- No drag-and-drop reordering yet
